# Fermionic Hamiltonian Engineering

Notebooks accompanying the paper on fermionic Hamiltonian engineering via local control.

## Structure

| Directory | Contents |
| --------- | -------- |
| [`Efficient_relaxation/`](Efficient_relaxation/README.md) | Efficient relaxation of the LP, classical vs quantum efficiency tradeoffs, comparing the relaxation to the optimal solution. |
| [`FHM-simulations/`](FHM-simulations/README.md) | Hamiltonian engineering for the 1D spinful Fermi-Hubbard model on a 6-site chain: energy and infidelity figures. |
| [`1088-mode-lattice/`](1088-mode-lattice/README.md) | Hamiltonian engineering on a 32×34 square lattice (1088 modes) targeting the Harper-Hofstadter model: Slater matrix dynamics and publication figures. |

See each subdirectory's README for dependencies, notebook descriptions, and expected outputs.
